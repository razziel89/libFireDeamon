Detailed Description

The class PG contains global information required for the parallelized computation.

The name stands for "ParallelGlobals". Parallelization is realized using multiple threads via pthreads. A mutex (stands for "mutually exclusive") for manipulating values in onjects of the class by all threads. A progress bar is also provided to allow printing progress reports.
