Detailed Description

A header containing template classes and function definitions that allow to perform parallelized computations.

This is achieved by mapping a function to every data set in a std::vector (with nigh-arbitrary template argument). The type bool is not supported as either input nor output type since std::vector<bool> is implemented as a bitfield and not as simply avector of Boolean values.
