Detailed Description

template<typename Tout, typename... Tins>
class GPSubData< Tout, Tins >

A templated class that contains all the data to be passed to single threads.

Multiple instances of this class are aggregated in GPData. GPSubData stands for "GenericParallelSubData". An arbitrary number of arguments can be passed to the single threads. Some bits for this class are taken from http://stackoverflow.com/questions/27941661/generating-one-class-member-per-variadic-template-argument
