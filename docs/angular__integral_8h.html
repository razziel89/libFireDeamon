Detailed Description

Contains classes that help in computing angular integrals that appear in pseudopotential integrals.

The algorithm that performs these computations is based on the following paper: Flores-Moreno, R., Alvarez-Mendez, R. J., Vela, A. and Köster, A. M. (2006), Half-numerical evaluation of pseudopotential integrals. J. Comput. Chem., 27: 1009–1019. doi:10.1002/jcc.20410
